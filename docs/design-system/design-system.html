6 Responsive Breakpoints

Chat Center использует 3 основных breakpoint-а. Принцип: mobile-first, progressive enhancement.

Breakpoint Range Layout Panels
Mobile ≤ 768px Single panel, full-width One panel at a time. List → Chat → Context via data-mobile-view. Back button visible. Bottom sheet for filters.
Tablet 769 – 1024px 2 panels List + Chat. Context hidden by default, shown as floating overlay on demand.
Desktop > 1024px 3 panels List + Chat + Context side by side. Sidebar visible.

Touch targets: minimum 44×44px on mobile. Input font-size: ≥ 16px to avoid iOS auto-zoom. Filter pills: horizontal scroll via .filters-scroll with overflow-x: auto.
