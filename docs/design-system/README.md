# AgentIQ Design System

## Components
- `HELP_PANEL.md` - единый паттерн контекстной панели (slide-out) для объяснений и onboarding внутри экранов.

## Principles
- Один визуальный язык компонентов для всех разделов (`Главная`, `Аналитика`, `Промокоды`, `Настройки`).
- Контекст рядом с действием: триггер открытия панели всегда ставится в верхнюю строку соответствующего блока.
- Без перегруза: в панели пишем простым языком, без длинных формул.
