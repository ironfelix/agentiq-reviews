# Guardrails Specs

Last updated: 2026-02-13
Status: active

Guardrails - это спеки, которые запрещают плохие ответы и небезопасные действия.
Они должны быть:
- проверяемыми (unit tests / lint checks),
- трассируемыми (где применяются в коде),
- и одинаково применяться к AI-draft и к фактической отправке (reply/publish).
