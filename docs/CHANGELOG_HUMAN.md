## 2026-02-17

### Новые возможности
- Единый inbox: отзывы, вопросы и чаты Wildberries теперь в одном списке — больше не нужно переключаться между вкладками
- ИИ-помощник автоматически составляет черновики ответов на каждое обращение
- Автоответы на позитивные отзывы (4–5★) без участия менеджера
- Приоритизация очереди: срочные обращения (дефект, неверный товар) выделены красным и стоят первыми
- Аналитика качества ответов: доля принятых, отредактированных и ручных ответов
- Ops-алерты: система сигнализирует о просроченных SLA и аномалиях в отзывах
- История переписки: все обращения одного покупателя по одному товару объединены в цепочку

### Надёжность и скорость
- Система стала стабильнее при одновременной работе нескольких менеджеров
- Синхронизация с Wildberries работает непрерывно в фоне — новые обращения появляются автоматически
- Страница состояния системы теперь загружается мгновенно (была ~15 секунд)
- Устранены периодические зависания фоновых задач при высокой нагрузке
- Авторизация стала безопаснее: обновлён алгоритм хранения паролей

### Интерфейс
- Фильтры "Срочно", "Без ответа", "Обработаны" в левой панели
- Мобильная версия: полноэкранный чат, свайп между панелями
- Индикатор синхронизации показывает когда данные обновляются
- Настройки вынесены в отдельную вкладку

## 2026-02-14

### Новые возможности
- Подключение магазина Wildberries через API-ключ — занимает 2 минуты
- Регистрация и вход без Telegram: email + пароль
- Черновики ответов: ИИ предлагает готовый текст, менеджер редактирует и отправляет одной кнопкой
- Guardrails: система не позволяет отправить ответ с запрещёнными фразами (обещания возврата, упоминание ботов)
- Timeline покупателя: история всех его обращений по всем товарам в одном месте
- Статус ответа на Wildberries: "ожидает подтверждения", "опубликован"

### Надёжность
- Инкрементальная синхронизация: загружаем только новые обращения, не всё с нуля
- Лимит запросов к API Wildberries: система автоматически соблюдает ограничения платформы
- Индексы базы данных: списки и фильтры стали работать в 5–10 раз быстрее
- Мониторинг: метрики синхронизации, 4 типа автоматических алертов для команды
- E2E тесты: ключевые сценарии проверяются автоматически при каждом обновлении

## 2026-02-10

### Новые возможности
- Анализ интента: система определяет тип обращения (дефект, возврат, вопрос о размере, предзаказный вопрос)
- SLA таймер: для каждого обращения показан дедлайн ответа
- Автозакрытие чатов: неактивные чаты закрываются через 10 дней автоматически
- Промокоды в настройках аккаунта
