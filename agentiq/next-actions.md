## 2. North Star на 4–6 недель

**MVP+ = единое окно чатов для 1 маркетплейса (Ozon), с paid-пилотами.**

**Почему Ozon:** самая понятная API-документация и быстрый доступ к реальным чатам.

---

## 3. Границы MVP+ (чтобы не расползтись)

**Включаем (must-have):**
- Подключение аккаунта Ozon (API key + Client ID).
- Синхронизация чатов (polling 60s).
- Список чатов, лента сообщений, отправка ответов.
- Базовые фильтры (непрочитанные, маркетплейс=Ozon).
- Таймер SLA / дедлайны ответа.
- Простая страница настроек + health/логирование.

**Не включаем (out-of-scope для MVP+):**
- Мульти-маркет (WB/Яндекс) — **после MVP+**.
- Webhooks и real-time WS — **после MVP+**.
- AI-автоответы — **после MVP+**.
- Полная CRM, аналитику трендов, RAG, сложные триггеры.

**Критерии успеха MVP+:**
- 3–5 продавцов подключили Ozon аккаунт.
- 1–2 платящих клиента (или подписанный пилот на оплату).
- Ежедневное использование (вход + ответы).
- Время ответа сокращено хотя бы на 30%.

## 4. Дорожная карта (обновлённая)

### Week 1 — Core MVP+
- БД таблицы: `chat_accounts`, `chats`, `chat_messages`, `chat_sync_state`.
- OzonConnector (list chats, history, send message).
- Celery task `sync_chats` (polling).
- API endpoints: connect account, list chats, get messages, send.
- Простой UI (list + окно чата).

### Week 2 — Полировка + пилоты
- Stable sync + deduplication.
- Базовые фильтры (unread, status).
- Логи + алерты ошибок (Sentry/лог файл).
- Onboarding doc: как подключить Ozon API.
- 3–5 пилотов, ручной запуск.

### Week 3–4 — От пилотов к оплате
- Пакеты/цены (минимум 1 тариф).
- Простейшая оплата (manual invoice/transfer) — чтобы не блокировать первые деньги.
- Case study + короткая посадочная страница «Chat Center».
- Расширить на WB или Яндекс (если нужно для сделки).

## 5. Что нужно, чтобы дойти до денег

**Минимум для первых оплат:**
- 1 работающий маркетплейс (Ozon).
- Видимость всех чатов + возможность ответить.
- Чёткая ценность: SLA + «всё в одном окне».
- Онбординг + техподдержка.

**Стратегия:**
- 5 пилотов → 2 оплативших → ретеншн → масштаб.
- Цена MVP+: 5–15k ₽/мес (порог для SMB-селлеров).
- Упаковать «экономию времени + снижение потерь» как ROI.
