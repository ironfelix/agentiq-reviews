## Стратегия

1. **Week 1–3 Discovery:** широкий лендинг → собрать 50–100 leads → calls → найти топ-боли
2. **Week 4–6 MVP:** pivot в winning нишу (скорее всего WB/Ozon) → строим backend + dashboard
3. **Week 7–8 Launch:** beta → первые платящие клиенты (цель 75–150k MRR)
