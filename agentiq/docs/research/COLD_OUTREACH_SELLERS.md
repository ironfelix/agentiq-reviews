# Cold Outreach: Поиск контактов селлеров WB

Руководство по поиску контактов продавцов Wildberries для холодных продаж AgentIQ.

---

## 1. ВКонтакте (основной канал)

### Процесс:
1. Поиск по названию магазина/бренда в ВК
2. Ищем группы/сообщества селлера
3. В описании группы: email, WhatsApp, Telegram
4. Если нет публичных контактов → ЛС админу группы

### Паттерны поиска:
- `"название магазина" site:vk.com`
- `wildberries "название бренда"`

---

## 2. Telegram (высокий отклик)

### Чаты/каналы селлеров:
- "Wildberries Sellers"
- "WB Ozon Продавцы"
- "Маркетплейсы"
- "Селлеры России"
- Админы часто топовые селлеры с большими оборотами

### Поиск конкретного селлера:
```
Google: "название магазина" telegram
Google: "название бренда" @
```

### Парсинг ответов на отзывы:
Селлеры иногда оставляют @username в `answer_text` через WBCON API.

---

## 3. ЕГРЮЛ → Прямой поиск (надежный метод)

### Алгоритм:
```
1. Карточка товара на WB → Название продавца (ООО/ИП)
2. egrul.nalog.ru → Поиск по названию компании
   ↓
   Получаем: ИНН, ФИО директора, юр. адрес, телефон (опционально)
3. GetContact/Numbuster → Проверка телефона из ЕГРЮЛ
4. Google: "ФИО директора" + "wildberries"
5. Email patterns: info@domain.ru, sales@domain.ru
```

### Ресурсы:
- [egrul.nalog.ru](https://egrul.nalog.ru) - официальная база ЕГРЮЛ/ЕГРИП
- GetContact (приложение) - определение владельца номера
- Numbuster (веб) - альтернатива GetContact

- Закрепленные комментарии
- О себе → ссылки на соцсети

### Поиск:
```

YouTube: "как продавать на wb"
YouTube: "название ниши" + "wildberries"
```

## 7. Автоматизация через WBCON API ⚡

**Преимущество AgentIQ**: У нас уже есть доступ к отзывам и ответам селлеров!

### Скрипт парсинга контактов:
```python
import re
from scripts.wbcon_client import get_feedbacks

def extract_contacts(answer_text):
    """Извлекает контакты из ответов селлеров."""
    contacts = {}

    # Telegram
    tg = re.findall(r'@[\w]+', answer_text)
    if tg:
        contacts['telegram'] = tg

    # Email
    email = re.findall(r'[\w\.-]+@[\w\.-]+\.\w+', answer_text)
    if email:
        contacts['email'] = email

    # Телефон (РФ)
    phone = re.findall(r'(\+7|8)[\s\(]?\d{3}[\s\)]?\d{3}[\s-]?\d{2}[\s-]?\d{2}', answer_text)
    if phone:
        contacts['phone'] = phone

    # VK
    vk = re.findall(r'vk\.com/[\w]+', answer_text)
    if vk:
        contacts['vk'] = vk

    return contacts if contacts else None

# Пример использования
feedbacks = get_feedbacks(nm_id=123456789)
for fb in feedbacks:
    if fb.get('answer_text'):
        contacts = extract_contacts(fb['answer_text'])
        if contacts:
            print(f"Найдены контакты у {fb['seller_name']}: {contacts}")
```

### Фильтры для приоритизации:
- Селлеры с >100 ответами на отзывы (активные)
- Рейтинг продавца >4.5
- Объем продаж >1000 шт/мес (через mpstats.io)

## 8. Холодный заказ (творческий подход)

### Процесс:
1. Заказываем товар у целевого селлера
2. В посылке часто вкладыши с контактами (для повторных заказов)
3. Получаем: Telegram, WhatsApp, промокоды, группы ВК

### Плюсы:
- 100% рабочие контакты
- Можешь начать разговор: "Я купил у вас товар..."
- Видишь качество упаковки/сервиса (инсайт для питча)

### Минусы:
- Требует затрат (но цена товара = цена лида)

## 10. GetContact для определения владельца

### Процесс:
1. Находим телефон компании через ЕГРЮЛ
2. Пробиваем через GetContact (мобильное приложение)
3. Видим, как номер сохранен у других: "Иван WB", "Директор Магазин"
4. Если ЛПР → звоним/пишем в WhatsApp/Telegram

## Оптимальная воронка поиска

### Быстрый путь (15 минут):
```
1. Название магазина → ВКонтакте (5 мин)
2. Название магазина → Telegram поиск (5 мин)
3. Если не нашли → ЕГРЮЛ → GetContact (5 мин)
```

### Глубокий поиск (1 час):
```
1. ВК + Telegram (10 мин)
2. ЕГРЮЛ + email паттерны + hunter.io (15 мин)
3. YouTube каналы (10 мин)
4. Avito/Юла (10 мин)
5. Парсинг ответов через WBCON API (15 мин)
```
