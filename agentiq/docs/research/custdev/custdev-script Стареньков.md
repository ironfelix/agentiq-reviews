## После разговора: что фиксировать

Не текст, а структуру:

| Поле | Что записывать |
|------|----------------|
| Главный страх | Его словами |
| Главный триггер | Что конкретно запускает проблему |
| Что делают руками | Текущий процесс |
| Чего не понимают | Слепые зоны |
| Где теряют контроль | Момент когда ситуация выходит из-под руки |
| Текущие расходы на это | Ассистент / сервисы / своё время — в рублях если скажут |
| Готовность платить | Да / нет / при каких условиях |

Через 5–7 разговоров появится повторяющаяся формулировка боли и первый кандидат на killer-agent.
