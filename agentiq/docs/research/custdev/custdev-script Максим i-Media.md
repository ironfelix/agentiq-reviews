# Custdev — Максим, i-Media

**Тип:** агентство, ведут селлеров на маркетплейсах
**Отдел:** продвижение внутри площадок (WB, Ozon)

---

## Блок 1. Контекст

**Кто отвечает за отзывы и вопросы:**
- На стороне клиента (селлера), не агентства
- Если клиент ответственно подходит — лучше знают свой продукт, чем базовые шаблонные ответы

**Позиция по отзывам:**
- Ответы должны быть уникальными, чтобы покупатели видели что магазин взаимодействует
- По большей степени агентство с этим не работает

**Инструменты:**
- Самовыкупы
- Функционал чатов — знает что есть чат у Озона

---

## Блок 2. Рейтинг и реклама

**Рейтинг влияет на стоимость рекламы** — небольшая теория от Максима:

- Позиция в выдаче зависит от множества факторов: рейтинг, цена, количество отзывов
- Выше рейтинг товара и больше количество отзывов → влияет на позицию товара в выдаче
- Для популярных товаров ставка в рекламе будет ниже

**Пример (гипотеза):**
- 100 vs 1000 отзывов — топ-10 в выдаче
- Для карточки со 100 отзывов: ~2000 руб CPM
- Для карточки с 1000 отзывов: ~500 руб CPM

> "Есть РК — карточка будет наверху, такого не будет и всё работает сложнее"
