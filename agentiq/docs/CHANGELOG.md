# AgentIQ Chat Center — Changelog

## 2026-02-10

### Новые возможности
- Запуск Chat Center — единый центр управления чатами с покупателями на маркетплейсах (WB, Ozon)
- AI-анализ каждого чата: определение намерения покупателя, приоритета и рекомендация ответа
- Приоритизация чатов по SLA: срочные (негатив, предпродажные вопросы), обычные, обработанные
- Синхронизация всех чатов с Wildberries (39 чатов вместо 12 — полная выгрузка с пагинацией)
- Привязка чатов к товарам: автоматическое определение товара по данным заказа (31 из 39 чатов привязаны)
- Закрытие и переоткрытие чатов с отслеживанием статуса
- Автоматическое закрытие неактивных чатов через 10 дней
- Распознавание предпродажных намерений (размер, наличие, совместимость) с повышенным приоритетом
- Умный запрос отзыва: когда покупатель доволен решением проблемы, AI предлагает попросить оставить отзыв на товар
- Кнопки действий для AI-рекомендации: «Использовать» (скопировать в поле ввода) и «Обновить» (перегенерировать ответ)

### Улучшения
- AI-рекомендации стали естественнее: убран шаблонный суффикс, добавлены разнообразные примеры для каждого типа обращения
- Умное обращение по имени: AI извлекает имя покупателя, распознает фамилии и отчества, не здоровается фамилией
- Реальные ссылки в рекомендациях: прямые ссылки на чаты WB и карточки товаров вместо заглушек
- AI-анализ запускается сразу после синхронизации, а не с задержкой в 2 минуты
- Обновление лендинга: реальные скриншоты продукта вместо CSS-макетов
- Слайдер с навигацией на лендинге для просмотра экранов продукта (свайп на мобильных)
- Новый логотип AGENT<IQ> в стиле дашборда
- Обновление цен: бесплатный аудит + сервис от 4 990 руб./мес
- Чистый B2B-дизайн лендинга без эмодзи-декораций
- Добавлена иконка сайта (favicon) с логотипом IQ

### Исправления
- Фильтр «Обработаны» теперь корректно показывает отвеченные и закрытые чаты
- Раздел «В работе» больше не показывает чаты, на которые уже ответили
- Устранено задвоение приветствия в AI-рекомендациях («Анна, здравствуйте! Анна, здравствуйте!»)
- При уже отправленном ответе вместо рекомендации отвечать снова показывается статус «Вы ответили»
- Рекомендация появляется заново, когда покупатель пишет повторно
- Корректное отображение изображений в сообщениях (вместо пустых сообщений — плейсхолдер «Изображение»)
- Исправлен горизонтальный скролл на мобильных устройствах
- Длинные названия товаров и сообщения корректно обрезаются в списке чатов
- Переключение между чатами во время AI-анализа больше не перебрасывает на другой чат
- Корректное различение «нет чатов вообще» и «нет чатов по фильтру» в пустом состоянии
- Исправлена ошибка Celery с event loop при синхронизации
- Исправлена адаптивная верстка лендинга для экранов 641-767px
