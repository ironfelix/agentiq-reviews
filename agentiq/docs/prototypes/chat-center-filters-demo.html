Демо: Быстрые фильтры

Переключайтесь между двумя вариантами дизайна фильтров:

Вариант 1: Кнопки

  • ✓ Визуально более заметные, четкие границы
  • ✓ Привычный паттерн для фильтров
  • ✓ Счетчик в бейдже выглядит естественно
  • ✗ Занимают больше места вертикально
  • ✗ При 4+ фильтрах может быть перенос на новую строку

Вариант 2: Табы с dotted underline

  • ✓ Чище, минималистичнее
  • ✓ Меньше визуального шума
  • ✓ Экономия вертикального пространства
  • ✓ Dotted underline на hover показывает интерактивность
  • ✗ Счетчик менее заметен
  • ✗ При 5+ табах может быть тесно

Иконка фильтра (воронка)

Добавлена иконка-воронка справа в поле поиска для расширенных фильтров:

  • ✓ CSS-only решение, не требует SVG
  • ✓ Hover эффект показывает кликабельность
  • ✓ Active состояние (черный фон) показывает, что фильтры применены
  • Клик может открывать выпадающее меню с продвинутыми фильтрами (дата, платформа, оценка)

Рекомендация

Вариант 2 (Табы) лучше подходит для вашего минималистичного дизайна:

  • Меньше визуального шума, как у Otveto
  • Экономия места для списка чатов
  • Dotted underline на hover — элегантное решение
  • Solid underline у активного таба — четкий индикатор

Вариант 1 (Кнопки) можно использовать, если:

  • Нужно подчеркнуть количество в каждой категории
  • Фильтры — основная функция (но у вас это очереди)
  • Целевая аудитория предпочитает более традиционный UI

Сравнение с Otveto

У конкурента фильтры реализованы как кнопки-теги:

  • Переключатель "Все / Только новые" с счетчиком в бейдже
  • Кнопки "Без вашего ответа", "Без ответа клиента"
  • Светло-серый фон кнопок

Ваше преимущество:

  • У вас уже есть очереди "В работе" / "Ожидают ответа" — это частично заменяет фильтры
  • Можно совместить: очереди + быстрые фильтры для дополнительной сортировки
  • Drag-and-drop между очередями — у Otveto такого нет

Следующие шаги

  1. Выбрать вариант дизайна фильтров (кнопки или табы)
  2. Определить набор фильтров:
    • Все — показать все чаты
    • Новые — непрочитанные сообщения
    • Без ответа — требуют ответа продавца
    • Срочные — urgent чаты (harmful, risky)
    • Опционально: Harmful, Risky, Good
  3. Реализовать логику фильтрации в JS
  4. Добавить выпадающее меню расширенных фильтров по клику на воронку
