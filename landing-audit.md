# Аудит лендинга AgentIQ — Рекомендации

**Цель:** Усилить продукт, сделать CTA мощнее, улучшить конверсию

---

## 🎯 КРИТИЧНЫЕ ПРОБЛЕМЫ

### 1. **CTA слабый и неясный**

**Проблема:**
- "Получить анализ" — непонятно что будет дальше
- "Анализ по артикулу" — звучит как один раз, а не как продукт
- Два CTA (бесплатный анализ + пилот) — размывает фокус

**Решение:**
```
HERO CTA: "Попробовать бесплатно 7 дней"
- Понятно: это SaaS, можно попробовать
- Urgency: ограничено временем
- Risk-free: бесплатно

АЛЬТЕРНАТИВА: "Записаться на демо (15 мин)"
- Для тех кто хочет сначала посмотреть
- Личный контакт = выше конверсия в оплату
```

---

### 2. **Нет чёткого value proposition**

**Проблема:**
Сейчас в hero: "Центр контроля коммуникаций Wildberries"
- Что это даёт? Зачем мне?
- Фокус на фиче (центр контроля), а не на бенефите

**Решение:**
```
ВАРИАНТ A (ROI фокус):
"Перестаньте терять деньги на коммуникациях Wildberries"
Подзаголовок: Контролируйте сроки, качество ответов и точки риска
в одном окне. Команда работает быстрее, покупатели довольнее.

ВАРИАНТ B (Pain фокус):
"Все коммуникации Wildberries под контролем"
Подзаголовок: Больше никаких просроченных негативов, потерянных
вопросов и вредных ответов. Одно окно для всей команды.

ВАРИАНТ C (Outcome фокус):
"+20% к конверсии на вопросах. 0 пропущенных негативов."
Подзаголовок: Центр контроля коммуникаций WB с приоритетами,
дедлайнами и AI-помощником.
```

---
